Projects

Here are some of my projects I've worked on in the past. Feel free to view the hosted site or the github for each project below.

Doorz

Doorz is a social media site designed for developers. New users can create an account, login, create posts with the option to add an image, and comment on posts. This project was built with HTML, CSS, SCSS, JavaScript, Node.js, Express, jQuery and PostgreSQL

Hosted Site Github

Mumford & Sons

This is my own take on a band website for Mumford & Sons. A user can browse the bands differnt albums and view details of each as well as leave the band feedback. This project was built with HTML, CSS, JavaScript, Node.js, Express, EJS, & socket.io.

Hosted Site Github

VGS

VGS is a website that allows the user to search through a myriad of video games. The data for each game is pulled from an API and populates the page programmatically. This project was built with HTML, CSS, JavaScript, jQuery
