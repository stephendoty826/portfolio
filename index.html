Projects

Here are some of the projects I've worked on. Feel free to view the hosted site or the github for each.

Boss Monster

Boss Monster is single player version of the popular card game of the same name. The user must build their dungeon to defeat heroes. This project was built with React.js, Redux, JSX, HTML, JavaScript, node.js, CSS, Sequelize, Flexbox, and Bootstrap.

Hosted Site Github

Pizzazz

Pizzazz is a pizza building app where a user can either build their own pizza creations or choose from a list of fan favorites. This project was built with HTML, CSS, JavaScript, React, & Redux.

Hosted Site Github

Mumford & Sons

This is my own take on a band website for Mumford & Sons. A user can browse the bands different albums, chat with other fans, and leave feedback. This project was built with HTML, CSS, JavaScript, Node.js, Express, EJS, & socket.io.

Hosted Site Github

VGS

VGS is a website that allows the user to search through a myriad of video games. The data for each game is pulled from an API and populates the page programmatically. This project was built with HTML, CSS, JavaScript, jQuery.

Hosted Site Github
